### 🚀 Sobre mí

Soy **Ingeniero en Informática** titulado, egresado del Instituto Politécnico Nacional " IPN UPIICSA", con más de 3 años de experiencia, apasionado por resolver problemas complejos de negocio mediante software bien diseñado. Mi enfoque no está en casarme con un lenguaje en particular, sino en aplicar sólidos principios de arquitectura (Clean Architecture, SOLID), patrones de diseño y buenas prácticas independientemente de la tecnología elegida.

- 💡 **Versatilidad Tecnológica:** Cuento con experiencia sólida navegando tanto en ecosistemas robustos de tipado estático (Java/TypeScript) como en entornos ágiles de desarrollo rápido (PHP/JavaScript).
- 🛠️ **Full-Stack Mindset:** Capaz de diseñar modelos de datos eficientes, estructurar APIs transaccionales seguras y dar vida a interfaces de usuario interactivas y optimizadas.
- 📈 **Ciclo de Vida Completo:** Mi experiencia administrando entornos locales en **Windows** y despliegues sobre **Linux** o contenedores Docker me permite entender el ciclo de vida del software de extremo a extremo.
- 🤝 **Estándares Profesionales:** Fiel creyente de la arquitectura limpia, el código autodocumentado, las pruebas automatizadas (Pest/JUnit) y el desarrollo colaborativo enfocado en resultados.

### 💻 Arquitectura y Microservicios Destacados

He desarrollado un **HUB (Portafolio)** modular y altamente desacoplado, impulsado por diversos motores en el backend. Te invito a explorar sus componentes principales:

| Microservicio | Tecnologías Base | Características Destacadas |
| :--- | :--- | :--- |
| **[Pipeline ETL Logiflow](Enlace-a-tu-repo-logiflow)** | Java 21, Spring Batch, Hilos Virtuales | Ingesta masiva asíncrona, tolerancia a fallos (SkipPolicy) y WAF simulado. |
| **[Motor Transaccional](Enlace-a-tu-repo-motor)** | Spring Boot 3.5, PostgreSQL | Propiedades ACID, bloqueo pesimista (`PESSIMISTIC_WRITE`) y swaps atómicos con `BigDecimal`. |
| **[IAM Core API](Enlace-a-tu-repo-iam)** | PHP 8.3, Laravel 13, JWT | Autenticación stateless, middlewares estrictos de RBAC y bitácora de auditoría inmutable. |
| **[Motor de Reportería](Enlace-a-tu-repo-reportes)** | Laravel 13, Dompdf | Flujos de memoria stateless (`php://temp`), Rate Limiting y Patrón Strategy polimórfico. |
